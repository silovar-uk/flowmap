# Flowmap

図形と囲みを直接操作しながら、業務の流れ、前後関係、補足情報を整理するローカルWebアプリです。

## v0.11.0

モードを切り替えず、キャンバス上でそのまま手を動かせる操作へ更新しました。使い方レクチャーは自動では開かず、上部の**使い方**を押した時だけ始まります。

## すぐ使える操作

- **空白をクリック**：その場所に処理を追加し、すぐタイトル入力
- **空白をドラッグ**：ボード全体を移動
- **図形をドラッグ**：自由に移動
- **図形を別の図形へ重ねる**：重ねた相手と接続
- **図形を矢印へ重ねる**：矢印の途中へ挿入
- **接続点をドラッグ**：接続先を細かく指定
- **ホイール**：拡大・縮小

空白を押してから3px以内で離した場合はクリックとして処理を作成し、それ以上動かした場合はボード移動として扱います。

図形を重ねている間は接続先を強調表示します。離すと矢印を作成し、2つの図形が重なったままにならない位置へ自動で整列します。すでに接続済みの場合は矢印を重複させません。

## 使い方レクチャー

レクチャーは初回起動時を含め、自動では表示しません。上部の**使い方**から必要な時だけ開始できます。

説明を読むだけではなく、実際の操作で進みます。

1. **＋処理**を押して図形を追加
2. 空白をクリックして、その場所に処理を追加
3. 2つの図形を重ねて接続
4. 右パネルで補足情報を編集
5. 保存と一覧の使い分けを確認

操作が必要な段階では、対象操作を完了すると自動で次へ進みます。`Esc`または「終了」でいつでも閉じられます。

## 保存とボード一覧

- 編集中のボードはIndexedDBへ自動保存
- 上部の**保存**から現在のボードへ名前を設定
- 上部の**一覧**から保存したボードを更新順で表示
- 一覧では、開く、名前変更、複製、削除が可能
- **新しいボード**は現在のボードを残したまま空のボードを作成
- 現在編集中のボードは「編集中」と表示

IndexedDBデータベース`flowmap`は次のストアを使います。

```text
flowmap
├─ boards
│  └─ board_xxx
│     ├─ name
│     ├─ state
│     ├─ createdAt
│     ├─ updatedAt
│     └─ schemaVersion
└─ meta
   └─ activeBoardId
```

旧v0.9の`boards/current`レコードと旧localStorageデータは、必要に応じてIndexedDBへ自動移行します。

## 全体を削除

**データ → 全体を削除**から、現在のボードにある図形、矢印、囲み、フェーズをすべて削除できます。

- ボード名と保存枠は残る
- 他の保存済みボードには影響しない
- 確認画面を経由する
- 実行後は元に戻せない

一覧内の「削除」は、保存枠そのものを削除する別の操作です。

## フローチャート記法

- **開始／終了**：角丸の端子形
- **処理**：長方形
- **判断**：ひし形
- **入出力**：平行四辺形
- **書類**：下辺が波形の書類形
- **矢印**：上下左右から相手に近い辺へ直角接続
- **分岐ラベル**：判断から出る矢印に「はい／いいえ」などを表示

上部の**図形**から種類を選んで追加できます。既存図形は、選択後の「図形」操作または右パネルから変更できます。

## キーボード

- `Enter`：選択した図形を編集
- `Tab`：下に処理を追加
- `Shift + Enter`：子の処理を追加して接続
- `Ctrl / Cmd + D`：複製
- `Delete`：削除
- `Space + ドラッグ`：図形の上からボードを移動
- `F`：選択項目または全体へズーム
- `Ctrl / Cmd + K`：検索
- `Ctrl / Cmd + Z`：元に戻す

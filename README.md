# Flowmap

図形を置き、つなぎ、まとめながら、仕事の流れを触って理解するローカルWebアプリです。

## v0.15.0 — まとめて、閉じる

v0.15.0では、複数の工程を位置を変えずに囲み、作業が終わった部分や大きなまとまりを視界から自然に退かせられるようにしました。

- 完了した図形全体を灰色表示
- `Shift＋クリック`で選択へ追加・解除
- 空白から範囲ドラッグして複数選択
- `Ctrl / Cmd＋A`で表示中の図形を全選択
- 選択した図形の位置を変えずに、外側へ囲みを作成
- 異なるフェーズの図形を誤って一つの囲みにしない
- 囲みを見出しだけの大きさへ折りたたみ
- 折りたたみ中も外部との矢印を囲み外周へ表示
- 件数ボタンから内部の工程名一覧を表示
- 一覧から工程を選ぶと、囲みを展開して対象へ移動

## 複数選択

PCでは次の2つを併用できます。

- **範囲ドラッグ**：範囲内に中心点がある図形をまとめて選択
- **Shift＋クリック**：選択へ1件ずつ追加・解除

`Shift＋範囲ドラッグ`では、現在の選択へさらに追加できます。

複数選択するとキャンバス下部へ操作バーが現れます。

```text
4件を選択　［囲みにする］［解除］
```

空白クリックは従来どおり新しい工程を入力します。ボード移動は`Space＋ドラッグ`または中ボタンドラッグです。タッチ端末では従来どおりドラッグでボードを移動します。

## 選択から囲みを作る

2件以上の図形を選び、**囲みにする**を押します。

- 図形の座標は変更しない
- 選択図形の外接範囲へ余白を加えて囲みを作る
- 元のフェーズを引き継ぐ
- 既存の囲みに属する図形も新しい囲みへ移せる
- 元の囲みが空になっても自動削除しない
- 囲み作成と所属変更を1回のUndoで戻せる

異なるフェーズに属する図形が混ざっている場合は作成を止め、同じフェーズ内で選び直すよう案内します。

## 囲みを折りたたむ

折りたたみ時は、囲み名、完了数、件数、展開ボタンだけを残します。

```text
┌ 企画・設計    3/5完了    5件  ＋ ┐
└──────────────────────┘
```

- 内部の図形と内部だけを結ぶ矢印は非表示
- 外部につながる矢印は囲みの外周へ一時的に接続
- 元の矢印データは変更しない
- 再展開すると元の図形間接続へ戻る
- 全工程が完了した囲みは全体を灰色へ寄せる

折りたたみ中の**○件**を押すと、工程名の一覧をポップアップ表示します。完了工程は灰色、期限超過工程には注意印を表示します。工程名を押すと囲みを展開し、その図形へ移動します。

## 完了表示

状態を**完了**へ変えると、図形全体を無彩色にします。

- 背景、枠、文字、タグ、進捗を灰色へ統一
- 選択中の青い輪郭は残す
- 矢印は灰色にせず、後続工程への流れを維持
- 経路強調と流れの再生対象には含める

## 流れを見る

図形を選ぶと、キャンバス上部から次を実行できます。

- **ここから先**：選択した工程以降を強調
- **ここまで**：選択した工程へ至る前工程を強調
- **前後**：つながる経路全体を強調
- **流す**：工程と矢印を順番に再生
- `Esc`：経路表示と再生を解除

## 作る・確認・見せる

- **作る**：図形追加、接続、複数選択、囲み作成を優先
- **確認**：期限、タグ、メモ件数、進捗を表示し、作成操作を隠す
- **見せる**：編集UIを隠し、タイトルと流れの説明に集中

表示モードはボードごとにIndexedDBへ保存します。

## 基本操作

- 空白クリック：仮の処理を置き、`Enter`で確定
- 範囲ドラッグ：複数選択
- `Shift＋クリック`：選択へ追加・解除
- `Space＋ドラッグ`：ボード移動
- 図形を近づける：方向を選んで接続
- 接続点をドラッグ：既存図形へ接続
- `F2`：図形名を編集
- `Tab`：下へ次工程を追加して接続
- `Ctrl / Cmd＋Enter`：右へ子工程を追加して接続
- `Ctrl / Cmd＋Z`：元に戻す
- `F`：選択項目または全体へズーム

## 保存と補足情報

編集内容はIndexedDBへ自動保存します。各図形には状態、期限、担当、タグ、場所、リンク、チェックリスト、概要メモ、追加メモ、編集履歴を保存できます。

JSON / YAMLによるバックアップと読み込み、PDF出力、複数ボード管理にも対応しています。

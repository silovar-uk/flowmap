# Flowmap

図形を置き、つなぎ、まとめながら、仕事の流れを触って理解するローカルWebアプリです。

## v0.16.0 — 狙ったときだけ置く

空白を触るたび工程が増える操作をやめ、追加したいときだけ明確な追加モードへ入るようにしました。

- 通常の空白クリックは選択解除のみ
- 空白ドラッグは範囲選択
- `Space＋ドラッグ`または中ボタンドラッグでボード移動
- Windows / Linuxは`Ctrl＋空白クリック`で処理を追加
- Macは`Cmd＋空白クリック`で処理を追加
- 修飾キーを押している間、実際の追加位置へ半透明カードを表示
- 通常は28px方眼へ吸着
- `Alt` / `Option`併用中は自由配置
- 既存工程と重なる場合は最寄りの空き位置へ移動
- 同距離なら右、下、左、上の順を優先
- 囲み内では同じ囲みとフェーズへ所属
- 囲み内に空きがなければ外へ飛ばさず追加を止める
- 折りたたみ中の囲みへは直接追加しない
- 追加直後にタイトル入力を開始

### キャンバスから追加する

```text
Windows / Linux: Ctrl＋空白クリック
Mac:             Cmd＋空白クリック
```

キーを押している間、カーソルが追加モードへ変わり、処理カードのプレビューが表示されます。

```text
        ┌──────────────┐
        │ 処理を追加     │
        └──────────────┘
```

クリック位置が既存工程と重なる場合、プレビューの時点で最も近い空き位置へ移動します。元位置から移動した場合は、細いガイド線で関係を示します。

`Alt`または`Option`を押している間だけ方眼吸着を解除します。キーを離すと、プレビューと追加モードは消えます。

## 複数選択

PCでは次の2つを併用できます。

- **範囲ドラッグ**：範囲内に中心点がある図形をまとめて選択
- **Shift＋クリック**：選択へ1件ずつ追加・解除
- **Shift＋範囲ドラッグ**：現在の選択へ追加
- **Ctrl / Cmd＋A**：表示中の図形を全選択

複数選択するとキャンバス下部へ操作バーが現れます。

```text
4件を選択　［囲みにする］［解除］
```

キャンバス追加を実行した場合は、既存の複数選択を解除し、新しい処理だけを選択します。

## 選択から囲みを作る

2件以上の図形を選び、**囲みにする**を押します。

- 図形の座標は変更しない
- 選択図形の外接範囲へ余白を加えて囲みを作る
- 元のフェーズを引き継ぐ
- 既存の囲みに属する図形も新しい囲みへ移せる
- 囲み作成と所属変更を1回のUndoで戻せる

異なるフェーズの図形が混ざっている場合は作成を止めます。

## 囲みを折りたたむ

折りたたみ時は、囲み名、完了数、件数、展開ボタンだけを残します。

```text
┌ 企画・設計    3/5完了    5件  ＋ ┐
└──────────────────────┘
```

- 内部の図形と内部だけを結ぶ矢印は非表示
- 外部につながる矢印は囲みの外周へ一時的に接続
- 元の矢印データは変更しない
- 再展開すると元の図形間接続へ戻る
- 全工程が完了した囲みは灰色表示

**○件**を押すと工程一覧を表示します。工程名を押すと囲みを展開し、対象工程へ移動します。

## 完了表示

状態を**完了**へ変えると、図形全体を無彩色にします。

- 背景、枠、文字、タグ、進捗を灰色へ統一
- 選択中の青い輪郭は残す
- 矢印は灰色にせず、後続工程への流れを維持
- 経路強調と流れの再生対象には含める

## 流れを見る

図形を選ぶと、キャンバス上部から次を実行できます。

- **ここから先**：選択した工程以降を強調
- **ここまで**：選択した工程へ至る前工程を強調
- **前後**：つながる経路全体を強調
- **流す**：工程と矢印を順番に再生
- `Esc`：経路表示と再生を解除

## 作る・確認・見せる

- **作る**：追加、接続、複数選択、囲み作成を優先
- **確認**：期限、タグ、メモ、進捗の確認を優先
- **見せる**：編集UIを隠し、流れの説明に集中

キャンバス追加モードは**作る**モードでのみ有効です。

## 基本操作

- 空白クリック：選択解除
- 範囲ドラッグ：複数選択
- `Shift＋クリック`：選択へ追加・解除
- `Space＋ドラッグ`：ボード移動
- `Ctrl / Cmd＋空白クリック`：その位置へ処理を追加
- `Ctrl / Cmd＋Alt / Option＋クリック`：自由配置で処理を追加
- 図形を近づける：方向を選んで接続
- 接続点をドラッグ：既存図形へ接続
- `F2`：図形名を編集
- `Tab`：下へ次工程を追加して接続
- `Ctrl / Cmd＋Enter`：右へ子工程を追加して接続
- `Ctrl / Cmd＋Z`：元に戻す
- `F`：選択項目または全体へズーム

スマートフォンでは、ヘッダーの**＋処理**と選択工程の四方向追加ボタンを使用します。

## 保存と補足情報

編集内容はIndexedDBへ自動保存します。各図形には状態、期限、担当、タグ、場所、リンク、チェックリスト、概要メモ、追加メモ、編集履歴を保存できます。

JSON / YAMLによるバックアップと読み込み、PDF出力、複数ボード管理にも対応しています。

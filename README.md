# Flowmap

図形を置き、つなぎ、仕事の流れを触って理解するローカルWebアプリです。

## v0.14.0 — 流れを触って理解する

v0.14.0では、完成した図を眺めるだけでなく、経路を強調し、流れを再生しながら確認できるようにしました。

- **ここから先**：選択した工程以降の経路を強調
- **ここまで**：選択した工程へ至る前工程を強調
- **前後**：選択した工程につながる経路をまとめて強調
- **流す**：矢印上を光点が移動し、工程を順番に再生
- 関係のない図形と矢印は薄く表示
- 循環や分岐があっても同じ工程を無限に再生しない
- `Esc`で経路表示と再生を解除

## 作る・確認・見せる

ヘッダーから、同じボードの見せ方を切り替えられます。

### 作る

図形と接続の作成を優先するモードです。

- カード上はタイトルと状態を中心に表示
- 選択した図形の四方へ新規追加ボタンを表示
- 接続点から既存図形へ矢印を引く操作も利用可能

### 確認

業務情報を確認するモードです。

- 期限、タグ、メモ件数、チェック進捗を表示
- 作成ボタンと接続ハンドルを隠し、誤編集を抑制
- 期限超過の工程を強調

### 見せる

会議や説明のための表示モードです。

- 左右パネル、追加ツール、ミニマップ、編集ボタンを非表示
- タイトルと流れを中心に表示
- 経路の強調と再生操作は残す
- **作る**または**確認**を押すと編集画面へ戻る

表示モードはボードごとにIndexedDBへ保存します。

## 四方から工程を追加

図形を選択すると、接続点より外側に4つの`＋`が表示されます。

```text
        ＋ 前工程
           ↓
前工程 ＋  図形  ＋ 次工程
           ↓
        ＋ 次工程
```

- **右・下**：選択中の図形 → 新しい図形
- **左・上**：新しい図形 → 選択中の図形
- 作成直後にタイトル編集を開始
- 図形追加と矢印追加を1回のUndoで戻せる
- 元の図形と同じフェーズ・囲みを引き継ぐ

小さい接続点は既存図形へ接続する操作、外側の`＋`は新しい工程を作る操作です。

## 基本操作

- **空白をクリック**：仮の処理を置き、入力して`Enter`で確定
- **空白をドラッグ**：ボードを移動
- **図形を近づける**：上下左右の方向を選んで接続
- **接続点をドラッグ**：既存図形へ接続
- **矢印へ重ねる**：矢印の途中へ挿入
- `F2`：選択した図形の名前を編集
- `Tab`：下に次の処理を追加して接続
- `Ctrl / Cmd + Enter`：右に子工程を追加して接続
- `Ctrl / Cmd + D`：複製
- `Ctrl / Cmd + Z`：元に戻す
- `F`：選択項目または全体へズーム

## 矢印

- 終端と線の中央に進行方向を表示
- 選択時に接続元を「元」、接続先を「先」と表示
- 判断図形から出る矢印には「はい」「いいえ」を自動設定
- 流れの再生中は対象の矢印を濃くし、光点を移動

## コンパクトなカード

```text
処理・入出力   184 × 88px
開始／終了     184 × 72px
判断           190 × 104px
書類           184 × 92px
```

ズームアウト時はさらに小さく表示し、全体の流れを優先します。

## 補足情報

各図形には右パネルから次の情報を保存できます。

- 状態
- 期限
- 担当
- 短い補足：最大240文字
- タグ：最大30件
- 場所
- 関連リンク
- チェックリスト
- 概要メモ：最大12,000文字
- 見出し付き追加メモ：最大30件
- 編集履歴

付箋上には必要な情報だけを表示し、長文は右パネルへ集約します。

## ボード管理

- 編集内容はIndexedDBへ自動保存
- ヘッダーのボード名から、一覧、新規作成、名前変更、複製、削除
- ボード名メニューは`Esc`で閉じる
- JSON / YAMLによるバックアップと読み込み
- PDF出力
- 現在のボード内容だけを全体削除

## 使い方

使い方レクチャーは自動起動しません。ヘッダーの**使い方**を押した時だけ、一時的な練習ボードで開始します。

練習中の操作は保存されず、終了すると元のボードと選択状態へ戻ります。

# Flowmap

付箋と囲みを直接操作しながら、業務の流れ、前後関係、補足情報を整理するローカルWebアプリです。

## v0.8.0

自由なホワイトボード操作を残しながら、従来のフローチャート記法を必要な場面で使えるようにしました。

## フローチャート記法

- **開始／終了**：角丸の端子形。フローの入口と出口だけに使う
- **処理**：長方形。作業、手続き、実行内容に使う
- **判断**：ひし形。条件によって流れが分かれる箇所に使う
- **入出力**：平行四辺形。入力、受付、取得、出力に使う
- **書類**：下辺が波形の書類形。申請書、案内、成果物などに使う
- **矢印**：図形の上下左右から、相手に近い辺へ直角に接続する
- **分岐ラベル**：判断から出る矢印には「はい／いいえ」などを表示できる

上部の「図形」ボタンから種類を選んで追加できます。既存の付箋は、選択後の「図形」操作または右側の補足パネルから変更できます。既存データは自動的に「処理」として扱います。

### 追加と編集

- ボードの空白をダブルクリック：その場に「処理」を追加し、すぐ入力
- 上部の「図形」：開始／終了、処理、判断、入出力、書類を選んで追加
- 付箋をダブルクリック／選択してEnter：その場でタイトル編集
- 編集中にEnter／Tab：下に次の処理を追加
- 編集中にShift+Enter：右に子の処理を追加し、自動で接続
- 選択中の図形から、種類、期限、担当、状態、タグ、メモを更新
- 詳細情報は右側の補足パネルで編集

### ドラッグ操作

- 空白、フェーズ内、囲み内の何もない場所をドラッグ：ボード全体を移動
- `Space`を押しながらドラッグ／中ボタンドラッグ：図形の上からでもボードを移動
- 図形をドラッグ：自由に移動
- 図形を囲みへドロップ：所属する囲みとフェーズを変更
- 図形を矢印の途中へドロップ：前後の矢印へ自動挿入
- 図形同士を重ねる：2枚を新しい囲みへまとめる候補を表示
- 図形の上下左右にある点を既存図形へドラッグ：接続
- 接続点を空白へドラッグ：接続された新しい処理を作成
- Shiftを押しながら空白へ接続：3枚の分岐を作成
- 選択した矢印の端点をドラッグ：接続元・接続先を付け替え
- 囲みの見出しをドラッグ：囲みと内部の図形をまとめて移動

### キーボード

- `Enter`：選択した図形を編集
- `Tab`：下に処理を追加
- `Shift + Enter`：子の処理を追加して接続
- `Ctrl / Cmd + D`：複製
- `Delete`：削除
- `Space + ドラッグ`：図形の上からボードを移動
- `F`：選択項目または全体へズーム
- `Ctrl / Cmd + K`：検索
- `Ctrl / Cmd + Z`：元に戻す

### 表示と記録

- ズーム倍率に応じて、図形の情報量を自動変更
- 遠景では形、近景ではタイトル・状態・期限・担当・タグを表示
- 選択中の図形は遠景でも内容と操作を展開
- 更新時刻を図形へ表示
- 図形ごとの変更履歴を記録
- JSON / YAML入出力、PDF出力、検索、ミニマップ、自動整列
- AI、ログイン、外部APIは不要

## ファイル構成

圧縮した単一ファイルを廃止し、役割ごとに分割しています。

```text
index.html
styles/
  base.css
  board.css
  panels.css
  flowchart.css
src/
  core.js
  migration.js
  render-board.js
  render-inspector.js
  actions-notes.js
  actions-layout.js
  interactions-drag.js
  interactions-connect.js
  events-ui.js
  events-fields.js
  flowchart.js
  events-main.js
```

ビルド工程はありません。HTML、CSS、JavaScriptを直接編集して確認できます。

## 公開

GitHub Pagesを`main`ブランチのルートから直接公開します。

1. **Settings → Pages**
2. **Build and deployment → Source**を**Deploy from a branch**
3. Branchを**main**、Folderを**/ (root)**
4. **Save**

公開URL：

`https://silovar-uk.github.io/flowmap/`

## ローカルで開く

```bash
python -m http.server 8000
```

`http://localhost:8000`を開きます。

## データ

データはブラウザのlocalStorageに`flowmap:v7`として保存されます。v0.8では各図形に`type`、各矢印に`label`を追加しています。旧データにこれらがない場合は「処理」と空ラベルで補完します。`flowmap:v4`〜`v6`などの互換データも、元の保存領域を削除せずに可能な範囲で移行します。

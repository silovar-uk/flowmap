# Flowmap

図形と囲みを直接操作しながら、業務の流れ、前後関係、補足情報を整理するローカルWebアプリです。

## v0.12.1

選択した図形を`F2`で名前編集できるようにしました。クリックで図形を選択してから`F2`を押すと、その図形上でタイトル編集を開始します。入力欄を編集中の場合はブラウザ標準のキー入力を優先し、ショートカットは割り込みません。

v0.12.0では、初見ユーザーの認知ウォークスルーをもとに、ボタンの優先順位、入力確定、接続方向、保存管理、使い方レクチャーを組み直しました。

中心にした考え方は、操作数を減らすことよりも、**実行する前に結果が見えること**です。

## ヘッダーとボード管理

- 現在のボード名をヘッダーに常時表示
- ボード名を押すと、保存一覧、新規作成、名前変更、複製、削除を表示
- 編集内容はIndexedDBへ自動保存
- 「保存」と「一覧」の別ボタンは非表示にし、ボード名メニューへ統合
- 保存状態は「保存中…／保存済み／保存失敗」で表示
- PDF、JSON、YAML、読み込み、全体削除は**書き出し**へ集約

## キャンバス上の追加ツール

追加操作はヘッダーからキャンバス左上へ移しました。

- **＋処理**：既定の処理を追加
- 右隣の矢印：開始／終了、処理、判断、入出力、書類を選択
- **囲み**：仕事のまとまりを追加
- **フェーズ**：大きな区切りを追加
- **整える**：現在の囲み、現在のフェーズ、ボード全体から範囲を選択

左の構造一覧と右の詳細パネルは、最初は閉じた状態で開始します。右パネルは図形、矢印、囲み、フェーズを選択した時だけ開き、選択を外すと閉じます。

## 空白クリックと仮図形

- 空白をクリックすると、その場所に薄い仮図形を表示
- 名前を入力して`Enter`を押した時だけデータとして確定
- 未入力のまま別の場所を押すと破棄
- `Esc`でも破棄
- `Shift + Enter`はタイトル内の改行
- マウスで8px以上、タッチで12px以上動かすとボード移動として扱う

誤クリックだけでIndexedDBへ不要な図形が追加されない構成です。

## 方向を見ながら接続

図形を別の図形へ近づけると、相手の上下左右に接続方向を表示します。

- 中央付近では接続を確定しない
- 上下左右のいずれかへポインターを動かすと、その方向を強調
- 確定後の図形位置を半透明の枠で表示
- 作成予定の矢印も事前表示
- 離した時に図形を配置して接続
- すでに2図形が接続済みの場合は矢印を重複させない
- 実行直後に「元に戻す」を表示

従来の上下左右の接続点からドラッグする操作も残しています。図形を矢印へ重ねた場合は、矢印の途中へ挿入します。

## タイトル編集とキーボード

- `F2`：選択した図形の名前を編集
- `Enter`：タイトル編集を開始／編集内容を確定
- `Shift + Enter`：タイトル内で改行
- `Tab`：現在の図形の下に次の処理を追加
- `Ctrl / Cmd + Enter`：接続された子の処理を追加
- `Ctrl / Cmd + D`：複製
- `Delete`：削除
- `Space + ドラッグ`：図形の上からボードを移動
- `F`：選択項目または全体へズーム
- `Ctrl / Cmd + K`：検索
- `Ctrl / Cmd + Z`：元に戻す

通常の`Enter`だけで勝手に次の図形が増えないように変更しています。

## 図形内と右パネルの役割

図形内のクイック操作は、使用頻度が高いものだけに絞りました。

- 図形種類
- 状態
- 複製
- 削除

期限、担当、タグ、場所、関連リンク、チェックリスト、メモ、履歴は右パネルで編集します。

## 使い方レクチャー

レクチャーは初回起動時を含め、自動では表示しません。ヘッダーの**使い方**を押した時だけ始まります。

レクチャー中は一時的な練習ボードへ切り替わります。

1. **＋処理**を押す
2. 空白へ仮図形を置き、名前を入力して確定
3. 接続方向を選びながら2図形をつなぐ
4. 右パネルを確認
5. ボード名メニューを確認

練習中の追加、移動、接続はIndexedDBへ保存されません。終了すると、開始前に開いていたボードと選択状態へ戻ります。

## 保存方式

IndexedDBデータベース`flowmap`を使用します。

```text
flowmap
├─ boards
│  └─ board_xxx
│     ├─ name
│     ├─ state
│     ├─ createdAt
│     ├─ updatedAt
│     └─ schemaVersion
└─ meta
   ├─ activeBoardId
   └─ uiV12Prepared
```

旧`boards/current`レコードと旧localStorageデータは、必要に応じてIndexedDBへ自動移行します。

## 全体を削除

**書き出し → 全体を削除**から、現在のボードにある図形、矢印、囲み、フェーズをすべて削除できます。

- ボード名と保存枠は残る
- 他の保存済みボードには影響しない
- 確認画面を経由する
- 実行後は元に戻せない

一覧内の「削除」は、保存枠そのものを削除する別の操作です。

## フローチャート記法

- **開始／終了**：角丸の端子形
- **処理**：長方形
- **判断**：ひし形
- **入出力**：平行四辺形
- **書類**：下辺が波形の書類形
- **矢印**：上下左右から直角接続
- **分岐ラベル**：判断から出る矢印に「はい／いいえ」などを表示

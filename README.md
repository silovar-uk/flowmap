# Flowmap

図形と囲みを直接操作しながら、業務の流れ、前後関係、補足情報を整理するローカルWebアプリです。

## v0.10.0

IndexedDBの自動保存を複数ボード管理へ拡張し、「保存」「一覧」「全体を削除」と、画面上で操作を学べるレクチャー機能を追加しました。

## 保存とボード一覧

- 編集中のボードはIndexedDBへ自動保存
- 上部の**保存**から、現在のボードへ名前を設定
- 上部の**一覧**から、保存したボードを更新順で表示
- 一覧では、開く、名前変更、複製、削除が可能
- **新しいボード**は、現在のボードを残したまま空のボードを作成
- 現在編集中のボードは「編集中」と表示
- ブラウザのタイトルにも現在のボード名を表示

IndexedDBデータベース`flowmap`は、次のストアを使います。

```text
flowmap
├─ boards
│  └─ board_xxx
│     ├─ name
│     ├─ state
│     ├─ createdAt
│     ├─ updatedAt
│     └─ schemaVersion
└─ meta
   ├─ activeBoardId
   └─ tutorialSeen
```

旧v0.9の`boards/current`レコードは「移行したボード」として引き続き利用できます。旧localStorageデータも、IndexedDBにボードが存在しない場合のみ自動移行します。

## 全体を削除

**データ → 全体を削除**から、現在のボードにある図形、矢印、囲み、フェーズをすべて削除できます。

- ボード名と保存枠は残る
- 他の保存済みボードには影響しない
- 確認画面を経由する
- 実行後は元に戻せない

一覧内の「削除」は保存枠そのものを削除する操作です。編集中のボードを削除した場合は、別の保存済みボードへ移動し、最後の1件なら新しい空ボードを作成します。

## 使い方レクチャー

初回起動時に、画面内の対象箇所を強調しながら6段階で案内します。

1. 処理・図形の追加
2. 空白ドラッグによるボード移動
3. 接続点からの矢印作成
4. 右パネルでの補足編集
5. ボード名の保存
6. 保存一覧の管理

一度終了すると自動表示は止まります。上部の**使い方**から何度でも再開できます。

## フローチャート記法

- **開始／終了**：角丸の端子形。フローの入口と出口だけに使う
- **処理**：長方形。作業、手続き、実行内容に使う
- **判断**：ひし形。条件によって流れが分かれる箇所に使う
- **入出力**：平行四辺形。入力、受付、取得、出力に使う
- **書類**：下辺が波形の書類形。申請書、案内、成果物などに使う
- **矢印**：図形の上下左右から、相手に近い辺へ直角に接続する
- **分岐ラベル**：判断から出る矢印には「はい／いいえ」などを表示できる

上部の「図形」ボタンから種類を選んで追加できます。既存の図形は、選択後の「図形」操作または右側の補足パネルから変更できます。

## 余白と密度

- ヘッダー、左ナビゲーション、右パネルの外周余白を少し拡張
- 一覧、タグ、状態表示は高密度のまま維持
- 右パネルの入力項目間を広げ、ラベルと入力欄のまとまりを明確化
- 囲みの見出しと図形の間、図形同士の縦横間隔を確保
- 「全体」表示では、図全体が画面端へ張り付かない余白を確保
- 狭い画面では余白とボタン表示を縮め、キャンバス領域を優先

## 追加と編集

- ボードの空白をダブルクリック：その場に「処理」を追加して入力
- 上部の「図形」：開始／終了、処理、判断、入出力、書類を選んで追加
- 図形をダブルクリック／選択してEnter：タイトル編集
- 編集中にEnter／Tab：下に次の処理を追加
- 編集中にShift+Enter：右に子の処理を追加し、自動接続
- 選択中の図形から、種類、期限、担当、状態、タグ、メモを更新

## ドラッグ操作

- 空白、フェーズ内、囲み内の何もない場所をドラッグ：ボード全体を移動
- `Space`を押しながらドラッグ／中ボタンドラッグ：図形の上からでもボードを移動
- 図形をドラッグ：自由に移動
- 図形を囲みへドロップ：所属する囲みとフェーズを変更
- 図形を矢印の途中へドロップ：前後の矢印へ自動挿入
- 図形同士を重ねる：2枚を新しい囲みへまとめる候補を表示
- 図形の上下左右にある点を既存図形へドラッグ：接続
- 接続点を空白へドラッグ：接続された新しい処理を作成
- Shiftを押しながら空白へ接続：3枚の分岐を作成
- 選択した矢印の端点をドラッグ：接続元・接続先を付け替え

## キーボード

- `Enter`：選択した図形を編集
- `Tab`：下に処理を追加
- `Shift + Enter`：子の処理を追加して接続
- `Ctrl / Cmd + D`：複製
- `Delete`：削除
- `Space + ドラッグ`：図形の上からボードを移動
- `F`：選択項目または全体へズーム
- `Ctrl / Cmd + K`：検索
- `Ctrl / Cmd + Z`：元に戻す

## 公開

GitHub Pagesを`main`ブランチのルートから直接公開します。

1. **Settings → Pages**
2. **Build and deployment → Source**を**Deploy from a branch**
3. Branchを**main**、Folderを**/ (root)**
4. **Save**

公開URL：

`https://silovar-uk.github.io/flowmap/`

## ローカルで開く

```bash
python -m http.server 8000
```

`http://localhost:8000`を開きます。

JSON／YAMLの書き出し・読み込みは引き続き利用できます。IndexedDBはブラウザ、端末、公開URLごとに保存されるため、別端末との同期にはJSON書き出しを使用してください。
